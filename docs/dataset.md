In order to conduct sensitivity analysis on the developed constitutive models, only model optimized model parameters will be needed, which are provided in the `dataset` folder. For further information about how the models were proposed and optimized, please refer to the [paper](https://onlinelibrary.wiley.com/doi/full/10.1002/eng2.70367) that is linked to this repo, and the [earlier paper](https://www.mdpi.com/2073-4360/15/22/4434) preceeding this research.
